## 🏁 Start
1. Crie o arquivo <code>.env</code> na raiz do projeto
2. Rode o comando
```bash
yarn dev
```

## 🐳 Starting with docker
1. Crie o arquivo <code>.env</code> na raiz do projeto
2. Rode o comando e espere subir o container
```bash
docker-compose up -d
```
